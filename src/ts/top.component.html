OSS Libraries

Golang

  • go-prompt: Building powerful interactive prompt in Go, inspired by python-prompt-toolkit.
  • kube-prompt: An interactive kubernetes client featuring autocomplete using go-prompt.
  • rtmp: Server implementation of RTMP 1.0 protocol in Go.
  • systracer: Yet another system call tracer written in Pure Go.
  • gosearch: Web crawler and Search engine in Golang.
  • rexecute: Re-execute a command when specified files are modified.

Python

  • Kobin: A small and statically-typed web web framework for python
  • Feedy: RSS Feed fetching framework for python
  • pandas-validator: Validation Library for pandas's DataFrame and Series.
  • WSGICLI: Command Line Interface for developing WSGI application..
  • wsgi-static-middleware: WSGI Middleware for serving static files.
  • wsgi-vmprof: WSGI Middleware for integrating vmprof.
  • outlier-utils: Utility library for detecting and removing outliers from normally distributed datasets
  • django-label-tag-attr: Add css classes and html tag attributes to django form's label.

JavaScript

  • react-native-focus-scroll: To detect which children are focused when scrolling for React Native.
  • Comet: NicoNico style tweet viewer for Desktop
  • ng2-card: Card view component for Angular2
  • mdnote: Markdown editor with React.js, Babel and Electron.

Others

  • jpandas: Implementation of Pandas (Python Data Analysis Library) on java.
  • pym: Simple python interpreter manager
